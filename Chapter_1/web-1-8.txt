The charset attribute specifies the character encoding for the HTML document.
The HTML5 specification encourages web developers to use the UTF-8 character set, which
covers almost all of the characters and symbols in the world!

meta name Element
Most of the meta elements use the name attribute to specify the type of information that’s being provided. Common values for the meta name attribute are author, description, and ­keywords. Here’s an example with an author value for a name attribute:
<meta name="author" content="John Dean">
The name attribute’s value specifies the type of thing that the content attribute’s value specifies.
