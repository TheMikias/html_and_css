  1.4 html tag
To implement an element for a web page, you’ll need to use tags. For example, if you want to implement a strong element 
(in order to put emphasis on the element’s content and display using boldface), surround the content with <strong> tags.
 Here’s how to implement a strong element for the word “very”:
 Caiden was <strong>very</strong> happy when her tooth finally came out.
